# 包功能说明

config 包提供了环境变量加载功能，专门用于在 Go 项目中自动从项目根目录加载 .env 文件。该包通过向上遍历目录树查找包含 go.mod 文件的目录来确定项目根目录，然后自动加载该目录下的 .env 文件到环境变量中。这种设计使得项目配置管理更加规范，避免了硬编码敏感信息，特别适用于需要区分开发、测试和生产环境配置的应用程序。

## 函数

```go
func LoadEnvFromRoot() error
```

LoadEnvFromRoot 函数尝试从项目根目录加载 .env 文件。项目根目录被定义为包含 go.mod 文件的最近祖先目录。函数会从当前工作目录开始向上遍历目录树，直到找到包含 go.mod 文件的目录，然后加载该目录下的 .env 文件。如果成功找到并加载 .env 文件，返回 nil；如果未找到 go.mod 文件或 .env 文件，返回相应的错误。该函数在包初始化时自动调用，确保环境变量在程序启动时就被正确加载。
